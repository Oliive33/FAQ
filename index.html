FAQ
  • Qu'est-ce que le HTML & le CSS ?

    HTML (HyperText Markup Language) est un langage utilisé pour structurer le contenu d'une page web, tandis que CSS (Cascading Style Sheets) est utilisé pour définir son apparence visuelle. En clair le fond et la forme !

  • Que voulez-vous dire par "JavaScript" ?

    JavaScript est un langage de programmation basé sur le texte utilisé à la fois côté client et côté serveur qui nous permet de rendre les pages web interactives.

  • Qu'est-ce que Figma ?

    Avec Figma, je crée des maquettes web interactives pour mes clients. Cela permet de voir et de tester l'aspect interactif du site dès les premières étapes de conception avant même de coder, pour avoir une ligne directrice et gagner du temps. Cela favorise une meilleure compréhension et une plus grande satisfaction client. Surtout c'est une passion !!!

  • Quel éditeur de texte, j'utilise ?

    J'utilise l'éditeur de texte VS Code pour coder les différents languages, HTML, CSS, JAVASCRIPT, etc... afin de mettre en forme les sites web et faire les tests nécéssaire.

  • Qu'est-ce que je veux dire par site web sur-mesure ?

    Mon travail consiste à concevoir et développer des sites personnalisés pour répondre aux besoins spécifiques de mes clients. Plutôt que d'utiliser des modèles préfabriqués, je crée des sites uniques en fonction de leurs préférences et objectifs. Comprendre les besoins et concevoir une structure de site adaptée. Je m'assure que le site est optimisé pour tous les appareils.
